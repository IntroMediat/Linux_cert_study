# Linux Cert Study

An interactive study app for the **LPI Linux Essentials** certification. 80 practice questions broken into focused 20-question segments, plus a 40-question exam simulator that matches the real exam format.

**[▶ Launch the app](https://YOUR-USERNAME.github.io/linux-cert-study/)**

> Replace `YOUR-USERNAME` above with your GitHub username once Pages is enabled.

## Features

- **4 study segments** of 20 questions each — study in sprints instead of one long marathon
- **Exam Simulator** — 40 random questions with a 65% pass threshold, mirroring the real cert
- **Inline explanations** for every question, shown after you submit
- **Flag for review** — mark tricky questions and filter down to just those
- **Missed / Flagged filters** — after submitting, jump straight to what you got wrong
- **Progress tracking** — per-segment rings, overall completion, and a passed-segments counter
- **Keyboard shortcuts** — press `1`–`5` to answer the question nearest your screen center
- Works fully **offline** — it's a single HTML file, no install, no build, no dependencies

## Run it locally

Just download `index.html` and double-click it. Any modern browser works.

## Topics covered

Shell basics, file permissions, the filesystem hierarchy, package management, users and groups, processes, networking and DNS, regular expressions, archiving and compression, cloud service models, and Linux distributions.

## Notes

Explanations were written from Linux fundamentals as a study aid. If you spot anything that conflicts with official courseware, open an issue.
